CSS 기초

html은 텍스트나 이미지, 표 같은 각 요소를 웹 문서에 넣는 것(홈페이지 뼈대)

css는 텍스트 색상이나 크기, 이미지 위치, 표 색상, 배치 방법 등 웹 문서의 디자인 요소를 담당

javascript는 정적인 홈페이지를 동적으로 만들어주는 프로그램
